MedMamba Pneumonia Detection

A State Space Model for Medical Image Classification

Project Overview

This project implements a novel Mamba-based architecture for detecting pneumonia from chest X-ray images using the PneumoniaMNIST dataset. The model combines the efficiency of state space models with convolutional neural networks for medical image analysis.

Model Architecture

The MedMamba model combines:

  • Data augmentation layers (RandomFlip, RandomRotation)
  • Patch embedding via Conv2D
  • 4 MambaBlocks with skip connections
  • Global average pooling and dense classifier
